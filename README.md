# Meeting-Room-Project

Um projeto simples em flask

## Funcionalidades
- [x] Cadastrar Reserva
- [x] Remover Reserva
- [x] Consultar Reserva
- [x] Consultar Reservas da Sala
- [x] Consultar Reservas do Usuario

## Install
```
 pip install -r requirements.txt
```

## Run
```
 flask --app app init-db
 flask --app app run
```
